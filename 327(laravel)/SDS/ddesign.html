      • Table: User

    Attribute DataType Description
    id int Id of User table (Primary key)
    user_name varchar(50) user_name of User name which hold user name and it is also used to log in to TO-LET.
    password varchar(50) password is user password to login in to TO-LET
    email varchar(50) email is user mail which hold user mailing address
    sign_in_status varchar(50) It will show user is log in or not
    phone int phone will hold user phone number

      Table: Houses

    Attribute DataType Description
    id int Id of Houses table (Primary key)
    house_price int house_price is hold houses rent etc value
    house_area varchar(50) house_area hold all the area where house located
    house_size int house_size hold the house sqft area
    house_floor int house floor hold ,which floor the house is rent for.
    house_side varchar(50) house_side provide which side the house is facing
    house_bed int house_bed hold number of bed, the house has
    house_bath int house_bath hold number of bath, the house has.

      Table: Admin

    Attribute DataType Description
    id int Id of Admin table (Primary key)
    admin_name varchar(50) admin_name of admin name which hold user name and it is also used to log in to TO-LET.
    password varchar(50) password is user password to login in to TO-LET
    email varchar(50) email is user mail which hold user mailing address

      Table: House_Area

    Attribute DataType Description
    id int Id of House_Area table (Primary key)
    user_id int This user_id is the the foreign key of from User table 'id'. It connects the relation with User table to House_Area table.
    house_id int This user_id is the the foreign key of from House table 'id'. It connects the relation with House table to House_Area table.
