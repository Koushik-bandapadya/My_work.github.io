7. Implementation plan:

Database: All the tables of the database will be created and those will be filled with initial data.By using fatching data API we will fatch real time data. Then, the system will insert data according to needs.

Android App: This will be implemented using code(java). it will interact with the database to insert, update and retrieve data. the app will also give a GUI for user interaction.In App there will be VR(Virtual reality) system, which will help user to find home easily.

Website:The website will come with the same or less features of the android app.

Hardware: The system will be made to support all the hardware requirements mentioned in the SRS. And GPS and phone sensors will also be used to give the system VR(Virtual reality) support.
